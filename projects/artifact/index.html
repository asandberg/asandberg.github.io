Artifact

Description

A bullet hell game made with C# and Microsoft XNA Game Library in 2012 as my graduation wort at Oulu Vocational School. The player's goal is to shoot at enemies while avoiding waves of bullets using simple controls.

My Role

I created all the assets and functionality in the game. The project provided interesting challenges in terms of optimization and memory usage due to the large number of game objects. Additionally, the visual aspect of bullet waves allowed interesting combinations of programming, maths, and creative skills.
